The GitHub MCP Agent (Obot-routed) is a fork of the original MCP AI agent, adapted for a testbed environment. It features a modified setup where the GitHub MCP server is hosted behind the Obot gateway, and the GitHub PAT is managed server-side. The toolset is configured to allow all permissions for access-control evaluation, and the agent runtime uses `langgraph.prebuilt.create_react_agent`. The setup involves installing dependencies, configuring environment variables, and running the agent via CLI or Streamlit UI.
